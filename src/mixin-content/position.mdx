Position mixins follow the following naming pattern:

<CodeBlock mixinName={`position`}>
  #position.position-type(position, vertical value, horizontal value);
</CodeBlock>

* The supported *position-type* options are: **absolute**, **relative** and **fixed**
* The *position* parameter options are **top-left**, **bottom-left**, **top-right**, **bottom-right** and **all**
* All *position* values default to **0**.

## Examples:

<CodeBlock mixinName={`position`}>
  #position.absolute(top-left);
</CodeBlock>

The single line of code above outputs the following 3 lines of CSS:

<CodeBlock mixinName={`position`}>
  position: absolute;
  top: 0;    
  left: 0;
</CodeBlock>

This mixin can easily be modified:

<CodeBlock mixinName={`position`}>
  #position.absolute(bottom-right, 10px, 5px);
</CodeBlock>

Which will output the following CSS:

<CodeBlock mixinName={`position`}>
  position: absolute;  
  bottom: 10px;  
  right: 5px;  
</CodeBlock>

If you need to specify all positions you can use the following:

<CodeBlock mixinName={`position`}>
  #position.absolute(all);
</CodeBlock>

Which will output the following CSS:

<CodeBlock mixinName={`position`}>
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
</CodeBlock>

You can change any of the position values if needed:

<CodeBlock mixinName={`position`}>
  #position.absolute(all, 10px, 2rem, 5%, 1em);
</CodeBlock>

Which will output the following CSS:

<CodeBlock mixinName={`position`}>
  position: absolute;
  top: 10px;
  right: 2rem;
  bottom: 5%;
  left: 1em;
</CodeBlock>

Relative positioning is supported:

<CodeBlock mixinName={`position`}>
  #position.relative(top-right, 0, 50%);
</CodeBlock>

Which will output the following CSS:

<CodeBlock mixinName={`position`}>
  position: relative;  
  top: 0;  
  right: 50%;  
</CodeBlock>

Fixed positioning is supported:

<CodeBlock mixinName={`position`}>
  #position.fixed(top-left, -20px, 100%);
</CodeBlock>

Which will output the following CSS:

<CodeBlock mixinName={`position`}>
  position: fixed;  
  top: -20px;  
  left: 100%;  
</CodeBlock>
