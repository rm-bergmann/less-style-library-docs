Position mixins follow the following naming pattern:

<CodeBlock mixinName={`position`}>
  #position.position-type(vertical-horizontal, vertical value, horizontal value);
</CodeBlock>

* The supported position-type options are: **absolute**, **relative** and **fixed**
* Vertical and horizontal parameter options are **top-left**, **bottom-left**, **top-right** or **bottom-right**
* First and Second position values both default to **0**.

## Examples:

<CodeBlock mixinName={`position`}>
  #position.absolute(top-left);
</CodeBlock>

The single line of code above outputs the following 3 lines of CSS:

<CodeBlock mixinName={`position`}>
  position: absolute;
  top: 0;    
  left: 0;
</CodeBlock>

This mixin can easily be modified:

<CodeBlock mixinName={`position`}>
  #position.absolute(bottom-right, 10px, 5px);
</CodeBlock>

Which will output the following CSS:

<CodeBlock mixinName={`position`}>
  position: absolute;  
  bottom: 10px;  
  right: 5px;  
</CodeBlock>


<CodeBlock mixinName={`position`}>
  #position.relative(top-right, 0, 50%);
</CodeBlock>

<CodeBlock mixinName={`position`}>
  position: relative;  
  top: 0;  
  right: 50%;  
</CodeBlock>

<CodeBlock mixinName={`position`}>
  #position.fixed(top-left, -20px, 100%);
</CodeBlock>

<CodeBlock mixinName={`position`}>
  position: fixed;  
  top: -20px;  
  left: 100%;  
</CodeBlock>
