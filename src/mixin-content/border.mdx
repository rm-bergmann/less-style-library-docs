There are 2 types of Border Mixins: Border Styles and Border Radius.
All Border Mixins follow a naming pattern.

## Border Style Mixins

<CodeBlock mixinName={`border`}>
  #border.border-style(position, border-width, border-color);
</CodeBlock>

### Parameter Options:

* The *border-style* options are: **solid**, **dotted** and **dashed**
* The *position* parameter is optional
* The *position* options are **top**, **bottom**, **left**, **right**, **vertical** and **horizontal**
* The *border-width* default value is **1px**
* The *border-color* default value is **black**

## Border Radius Mixins

<CodeBlock mixinName={`border`}>
  #border.radius(position, value);
</CodeBlock>

### Parameter Options:

* The *position* parameter is optional
* The *position* options are **top**, **bottom**, **left** and **right**
* The default value is *4px*

---

## Usage Examples:

Black solid 1px border (default)

<CodeBlock mixinName={`border`} number={`01`}>
  #border.solid();
</CodeBlock>

If you want a 4px solid red border you can pass in the border width of 4px, and the desired color.

<CodeBlock mixinName={`border`} number={`02`}>
  #border.dotted(4px, #f00);
</CodeBlock>

If you want a 2px green top border only you pass in top (or bottom / left / right), border-width value and color

<CodeBlock mixinName={`border`} number={`03`}>
  #border.dashed(top, 2px, green);
</CodeBlock>

If you need paralell top & bottom or left & right borders pass in vertical or horizontal:

<CodeBlock mixinName={`border`} number={`04`}>
  #border.solid(vertical, 4px, #333);
</CodeBlock>

<CodeBlock mixinName={`border`} number={`05`}>
  #border.solid(horizontal, 2px, black);
</CodeBlock>

Add a 6px black left border

<CodeBlock mixinName={`border`} number={`06`}>
  #border.solid(left, 6px, black);
</CodeBlock>

Useful border-radius mixins when we need a top left and right radius or bottom left and right radius.

<CodeBlock mixinName={`border`} number={`07`}>
  #border.radius();
</CodeBlock>

<CodeBlock mixinName={`border`} number={`08`}>
  #border.radius(20px);
</CodeBlock>

<CodeBlock mixinName={`border`} number={`09`}>
  #border.radius(top);
</CodeBlock>

<CodeBlock mixinName={`border`} number={`10`}>
  #border.radius(bottom, 8px);
</CodeBlock>

<CodeBlock mixinName={`border`} number={`11`}>
  #border.radius(left, 10px);
</CodeBlock>

<CodeBlock mixinName={`border`} number={`12`}>
  #border.radius(right, 50px);
</CodeBlock>
