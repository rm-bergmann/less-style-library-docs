## Colors

<div className="code-block">
  <CodeBlock>
    #display.colors(@color, @background-color);
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  To minimize common repeatition, this mixin takes color as the first parameter and 
  background-color as the second parameter. By default, color is set to black and 
  background color set to white.
</div>

<div className="code-block">
  <CodeBlock>
    #display.colors();
  </CodeBlock>

  <div className="display display--08">Text Colour</div>
</div>

<div className={`code-block__description`}>
  To customize the values pass in the color and background-color of choice which will
  replace the defaults. In this case, the @color[secondary] value is set to the 
  background-color.
</div>

<div className="code-block">
  <CodeBlock>
    #display.colors(white, @color[secondary]);
  </CodeBlock>

  <div className="display display--09">Text Colour</div>
</div>

<hr />

## Dimensions
<div className="code-block">
  <CodeBlock>
    #display.dimensions(@type, @width, @height);
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  Use this mixin to add width and height values to elements. 
  The type parameter is optional, by default width is set to 100px and height
  is set to auto.
</div>

<div className="code-block">
  <CodeBlock>
    #display.dimensions();
  </CodeBlock>

  <div className="display display-secondary display--10" />
</div>

<div className={`code-block__description`}>
  To customize the values, pass in the desired width value as the first paramater 
  and height value as the second parameter. 
</div>

<div className="code-block">
  <CodeBlock>
    #display.dimensions(200px, 50px);
  </CodeBlock>

  <div className="display display-secondary display--11" />
</div>

<div className={`code-block__description`}>
  If the width and height are the same values pass in the type 'equal', then the 
  desired value (default is set to 100px)
</div>

<div className="code-block">
  <CodeBlock>
    #display.dimensions(equal, 120px);
  </CodeBlock>

  <div className="display display-secondary display--12" />
</div>

## Width

<div className={`code-block__description`}>
  Width displays can be used when you need a width, min-width and 
  max-width. By default width is set to auto, min-width is set to 
  auto and max-width is set to 100%.
</div>

<div className="code-block">
  <CodeBlock>
    #display.width(@width, @min, @max);
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  To set a min-width and max-width only, use the minmax type as 
  the first parameter. Min and Max values are the same as the 
  default mixin.
</div>

<div className="code-block">
  <CodeBlock>
    #display.width(minmax, @min, @max);
  </CodeBlock>
</div>

## Clearfix

<div className={`code-block__description`}>
  When floating an element's children left or right, apply this mixin so the parent 
  can retain it's height:
</div>

<div className="code-block">
  <CodeBlock>
    #display.clearfix();
  </CodeBlock>

  <div className="display display--13">
    <div className="display__child display__child-left"></div>
    <div className="display__child display__child-right"></div>
  </div>
</div>

## Space

<div className="code-block">
  <CodeBlock>
    #display.space(@type, @margin, @padding);
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  By default, this mixin sets margin and padding to 0.
</div>

<div className="code-block">
  <CodeBlock>
    #display.space();
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  To adjust the values, pass in a margin as the first parameter and padding as 
  the second parameter. In the example below, margin is set to 10px, and padding 
  is set to 1rem.
</div>

<div className="code-block">
  <CodeBlock>
    #display.space(10px, 1rem);
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  When both margin and padding values should be set to 0, you can pass in 0 as the
  only parameter. Note this is teh same as the default settings but just slightly 
  more readable. 
</div>

<div className="code-block">
  <CodeBlock>
    #display.space(0);
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  When margin and padding have the same values, use the 'equal' type as the first 
  parameter, and pass in a value (defaults to 0). 
</div>

<div className="code-block">
  <CodeBlock>
    #display.space(equal, 1rem);
  </CodeBlock>
</div>

## Pseudo Element

<div className={`code-block__description`}>
  When targeting the ::before and ::after pseudo elements apply this mixin to 
  display the Pseudo element with the content value. By default the following
  styles are added:

  <div className="code-block">
    <CodeBlock>
      {`display: block; 
content: '';`}
    </CodeBlock>
  </div>
</div>

<div className="code-block">
  <CodeBlock>
    #display.pseudo(@content, @type);
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  In this example below the ::after pseudo element was targeted, and a 
  right angle quote character has been placed inside of it. Additionally, 
  a width and height value was added with a mixin. 
</div>

<div className="code-block">
  <CodeBlock>
    {`#display.pseudo('â€º', inline);
#display.dimensions(equal, 16px)`}
  </CodeBlock>

  <div className="display display--14"><span>Some Text</span></div>
</div>
