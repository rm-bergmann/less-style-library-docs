---
title: Form
slug: form
category: mixins
description: Improves default browser form input, label, select and textarea styles
---

import CodeBlock from '../../components/codeBlock/codeBlock';
import FormSelect from '../../components/formSelect/formSelect';

<CodeBlock mixinName={`form`}>
  #form.form-element(options);
</CodeBlock>

* Useful to use as a starting point when designing the element styles
* Form elements currently supported are **label**, **input**, **select** and **textarea**
* Options may vary depending on the form element

---

## Label

Browser default label:

<div className="form-container">
  <label>Default Label Style</label>
</div>

Label Mixin Style:

<CodeBlock mixinName={`form`}>
  #form.label();
</CodeBlock>

<div className="form-container">
  <label className="form-examples--label" htmlFor="text-input">
    Form Label
  </label>
</div>


<CodeBlock mixinName={`form`}>

</CodeBlock>

<div className="form-container">

</div>

## Select

Browser default select:

<FormSelect />

Select Element default style:

<FormSelect selectClassName="form-select--default" />

<CodeBlock mixinName={`form`}>
  #form.select();
</CodeBlock>
      

### Select Element style with radius:

<FormSelect selectClassName="form-select--rounded" />

<CodeBlock mixinName={`form`}>
  #form.select(rounded);
</CodeBlock>

## Textarea

Browser default textarea styles are plain and unatractive:

<textarea  />

Using the mixin below, you can instatly size it so it's more visible

<CodeBlock mixinName={`form`}>
  #form.textarea();
</CodeBlock>

<div className="form-container">
  <textarea className="form-textarea" />
</div>
