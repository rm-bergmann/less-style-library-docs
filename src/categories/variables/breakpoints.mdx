import CodeBlock from '../../components/codeBlock/codeBlock';

<div className={`code-block__description`}>
  Common device screen sizes are set up as variables to make it easier and
  faster to write media queries. The variables follow a specific pattern
  to make the variables easy to remember.
</div>

Breakpoint variables are constructed based on the following pattern:

<div className="code-block">
  <CodeBlock mixinName={`breakpoints`}>
    @media @[device]-[size]-[min/max]
  </CodeBlock>
</div>

## Device Options:
* mobile
* tablet
* laptop
* desktop
   
## Size Options
* small
* large

### Examples:

<div className={`code-block__description`}>
  To target a screen width of 319px and below: 
</div>

<div className="code-block">
  <CodeBlock mixinName={`breakpoints`}>
    @media @mobile-small-max
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  Which will output the following CSS:
</div>

<div className="code-block">
  <CodeBlock mixinName={`breakpoints`}>
    @media only screen and (max-width: 319px)
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  To target a viewport width of 1280px and above: 
</div> 

<div className="code-block">
  <CodeBlock mixinName={`breakpoints`}>
    @media @desktop-large-min
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  Which will output the following CSS:
</div>

<div className="code-block">
  <CodeBlock mixinName={`breakpoints`}>
    @media only screen and (min-width: 1280px)
  </CodeBlock>
</div>

<div className={`code-block__description`}>
  To target mobile devices with a screensize less than 320px wide you can
  write the following:
</div>
